# Task Manager - Менеджер Завдань

Простий та інтуїтивний веб-додаток для управління завданнями з підтримкою пріоритетів, фільтрації та пошуку.

## Можливості

- Додавання, редагування та видалення завдань
- Встановлення пріоритетів (низький, середній, високий)
- Пошук завдань
- Статистика виконання
- Автоматичне збереження в LocalStorage
- Експорт даних у TXT формат
- Сучасний та адаптивний дизайн

## Структура проекту

```
task-manager/
│
├── index.html          # Головна HTML сторінка
├── styles.css          # Стилі додатку
├── app.js              # JavaScript логіка
├── package.json        # Метадані проекту
├── .gitignore          # Git ignore файл
└── README.md           # Документація
```

## Як запустити проект

### Варіант 1: Відкриття файлу безпосередньо

1. Перейдіть до папки проекту `e:/LAB/task-manager/`
2. Двічі клацніть на файл `index.html`
3. Додаток відкриється у вашому браузері за замовчуванням

### Варіант 2: Через провідник

1. Відкрийте провідник Windows
2. Перейдіть до `E:\LAB\task-manager\`
3. Клацніть правою кнопкою миші на `index.html`
4. Виберіть "Відкрити за допомогою" → Ваш улюблений браузер

### Варіант 3: Через локальний сервер (рекомендовано для розробки)

Якщо у вас встановлено Node.js:

```bash
# Встановіть простий HTTP сервер
npm install -g http-server

# Перейдіть до папки проекту
cd e:/LAB/task-manager

# Запустіть сервер
http-server

# Відкрийте браузер за адресою http://localhost:8080
```

Або використовуйте Python:

```bash
# Python 3
cd e:/LAB/task-manager
python -m http.server 8000

# Відкрийте браузер за адресою http://localhost:8000
```

## Як користуватися

1. **Додати завдання**: Введіть текст у поле вводу, виберіть пріоритет та натисніть "Додати завдання"
2. **Позначити як виконане**: Клацніть на чекбокс біля завдання
3. **Видалити завдання**: Натисніть кнопку "Видалити" біля завдання
4. **Фільтрувати**: Використовуйте кнопки "Всі", "Активні", "Виконані"
5. **Шукати**: Введіть текст у поле пошуку
6. **Експортувати**: Натисніть "Експортувати дані" для збереження у TXT
